# Windows Server Infrastructure Lab

This project is a personal cybersecurity and infrastructure lab where I build a complete Windows Server environment from scratch.

The goal is to simulate a real corporate network environment while practicing system administration and security implementation.

## Technologies

- Windows Server
- Active Directory
- Group Policy (GPO)
- DNS
- DHCP
- Print Server
- Network Policy Server (RADIUS)

## Part 1 - Active Directory Setup

Implemented:

- Windows Server installation
- Active Directory Domain Services
- User and administrator creation
- Security Group Policy
- USB removable device blocking

## Part 2 - Print Server

Planned:

- Printer deployment
- Network printer management
- Print permissions

## Part 3 - Network Authentication

Planned:

- NPS (RADIUS)
- Network authentication policies
- Access control implementation

## Objective

Improve skills in:

- Infrastructure
- System administration
- Security hardening
- Enterprise environment simulation
